This two-column layout uses unit-25 for the sidebar and unit-75 for the content area. On tablet-sized screens and below, both columns stack to full width using tablet-unit-100.

The grid is built on flexbox with a units-row wrapper. Units automatically collapse to 100% width at the tablet breakpoint unless overridden with responsive classes.
